Hentai@Home (H@H) のバージョン更新、毎日のアーカイブクォータ、およびすべてのクライアントのオンライン/オフラインステータスを監視するための軽量な Python スクリプトです。

### 特徴
- **🚀 リアルタイム監視**: 5分ごと（設定可能）にステータスの変化を確認します。
- **📊 クライアントステータス**: IP、バージョン、信頼、ヒット率などの詳細なステータスレポート。
- **📅 クォータ追跡**: 無料アーカイブクォータの毎日の通知。
- **🔔 多チャンネル通知**: Bark, Telegram, PushPlus, PushDeer, Pushover, Discord, Slack, Gotify, Matrix, DingTalk をサポート。
- **🌐 多言語対応**: 英語、中国語（簡体字/繁体字）、日本語、韓国語、スペイン語、フランス語、ロシア語、ドイツ語、アラビア語、ヘブライ語の通知をサポート（全11言語）。
- **🐳 Docker サポート**: GitHub Container Registry (GHCR) でビルド済みイメージを利用可能。
- **🛡️ ゼロ依存**: Python 3.11+ 標準ライブラリのみで構築。
