Un script Python léger pour surveiller les mises à jour de version de Hentai@Home (H@H), le quota d'archive quotidien et le statut en ligne/hors ligne de tous vos clients.

### Caractéristiques
- **🚀 Surveillance en temps réel**: Vérifie les changements toutes les 5 minutes (configurable).
- **📊 Statut du client**: Rapports détaillés incluant l'IP, la version, la confiance, etc.
- **📅 Suivi du quota**: Notifications quotidiennes de votre quota d'archive gratuit.
- **🔔 Multi-canal**: Support pour Bark, Telegram, PushPlus, PushDeer, Pushover, Discord, Slack, Gotify, Matrix, DingTalk et Webhooks génériques.
- **🌐 Multilingue**: Support de notification pour 11 langues.
- **🐳 Support Docker**: Images pré-construites disponibles sur GHCR.
