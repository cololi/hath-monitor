Hentai@Home (H@H) 버전 업데이트, 일일 아카이브 쿼터 및 모든 클라이언트의 온라인/오프라인 상태를 모니터링하기 위한 경량 Python 스크립트입니다.

### 주요 기능
- **🚀 실시간 모니터링**: 5분마다(설정 가능) 상태 변화를 확인합니다.
- **📊 클라이언트 상태**: IP, 버전, 신뢰도, 히트레이트 등을 포함한 상세 상태 보고.
- **📅 쿼적 추적**: 무료 아카이브 쿼터에 대한 일일 알림.
- **🔔 다양한 알림 채널**: Bark, Telegram, PushPlus, PushDeer, Pushover, Discord, Slack, Gotify, Matrix, DingTalk 지원.
- **🌐 다국어 지원**: 영어, 중국어(간체/번체), 일본어, 한국어, 스페인어, 프랑스어, 러시아어, 독일어, 아랍어, 히브리어 알림 지원 (총 11개 언어).
- **🐳 Docker 지원**: GitHub Container Registry (GHCR)를 통한 사전 빌드 이미지 제공.
- **🛡️ 의존성 없음**: Python 3.11+ 표준 라이브러리만 사용.
